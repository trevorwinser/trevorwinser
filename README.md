# Hi, I'm Trevor Winser
### A passionate Software Developer

## Solo Projects
1. [Fall Through](https://pixelstolookat.itch.io/fall-through)
  A platformer played in reverse, with the princess using a bugged physics engine to escape the castle to reach her knight.
1. [Cowboy Game](https://github.com/trevorwinser/CowboyGame)
  My first game was an endless wave shooter starring a cowboy shooting zombies.
1. [Deduction](https://github.com/trevorwinser/Deduction)
  An online board game about solving a murder by making guesses with various clues. As of Jan 14, 2026, the game can be played [here](https://deduction-production.up.railway.app/).
1. Many Minecraft Games
  These were created for friends during COVID and throughout my university years. Through these games, I learned the value of player feedback and thorough playtesting.
